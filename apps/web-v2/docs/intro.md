### Weibo Archiver: 微博备份工具

#### 1. 为什么需要备份微博？

微博承载着我们的生活记录和回忆，但这些数据可能因为各种原因丢失：账号被封、内容被删除、或者平台政策变化。官方的备份功能有限，而且数据不在自己手里总是不安心。

**Weibo Archiver** 就是为了解决这个问题而生的。它不只是简单的导出工具，而是一套完整的微博备份和浏览解决方案。

**核心理念：你的数据，你做主。**

#### 2. 主要功能

- **全面备份**: 支持微博文字、图片、评论、转发、关注列表、收藏夹等所有内容的备份
- **多种工具**: 提供油猴脚本、浏览器插件、命令行工具，适合不同需求的用户
- **美观浏览**: 提供现代化的网页查看器，让你舒适地浏览备份的微博
- **本地存储**: 所有数据保存在本地，保护隐私，支持离线访问
- **开源免费**: 基于 Apache-2.0 协议开源，完全免费使用

#### 3. 使用指南

**🎯 推荐方式：浏览器插件 + 网页端**

这是最推荐的使用方式，可以实现自动化、增量备份，体验最佳。

1.  **安装浏览器插件**:
    - 前往 [Chrome 插件商店](https://chrome.google.com/webstore/category/extensions) / [Firefox Browser ADD-ONS](https://addons.mozilla.org/firefox/) 安装 **Weibo Archiver** 插件。（*注：插件正在审核中，此处为占位链接*）

2.  **打开网页端引导页**:
    - 安装插件后，访问我们的[在线网页端](https://weibo-archiver.chilfish.top/sync)的同步页面。

3.  **开始备份**:
    - 页面会自动检测插件是否安装。
    - 你只需在输入框中填入你想要备份的微博用户 UID（通常是你个人主页 URL 中的一串数字），然后点击“开始备份”即可。
    - 插件将在后台为你执行所有抓取任务，并实时将进度同步到网页上。

4.  **浏览你的备份**:
    - 备份完成后，你可以随时访问[网页端主页](https://weibo-archiver.chilfish.top/post)，所有数据都已保存在你的浏览器中，供你随时浏览、搜索和查看。

**📥 其他数据获取方式**
- **油猴脚本**：使用插件的替代方案，适合一次性备份，操作简单
- **命令行工具**：适合批量处理和自动化场景

所有工具获取的数据都可以导入到网页端查看器中统一浏览。

> 详细使用方法请参考各工具的专门文档：[浏览器插件](./extension.md) | [网页端](./web.md) | [本地服务器](./server.md) | [油猴脚本](./monkey.md) | [命令行工具](./cli.md)

#### 4. 版本迁移

如果你之前使用过旧版本 (`weibo.chilfish.top`)，迁移很简单：在 [新版查看器](https://weibo-archiver.chilfish.top/) 中重新导入你的 `weibo-data.json` 文件即可。

注意：新版油猴脚本与旧版微博数据兼容，但关注列表、收藏夹等需要重新导出。
